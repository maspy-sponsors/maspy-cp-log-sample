# maspy-cp-log sample

このレポジトリは、GitHub Sponsors の有料スポンサー向けに限定公開している
`maspy-cp-log` の公開サンプルです。

## 内容

- `log/`: 完全版から選んだ問題の提出コードと解法メモ
- `index/`: サンプル収録問題だけを対象に生成した各種index
- `full-repository/`: 完全版レポジトリのREADME・統計・最近の追加状況

`full-repository/` 以下の次のファイルは、完全版レポジトリから自動同期する予定です。

- [`full-repository/README.md`](full-repository/README.md)
- [`full-repository/README_en.md`](full-repository/README_en.md)
- [`full-repository/index/recent.md`](full-repository/index/recent.md)
- [`full-repository/index/stats.md`](full-repository/index/stats.md)

サンプルとして収録する問題と、その問題を対象にした `index/` は自動同期しません。
